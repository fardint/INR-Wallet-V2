Our road map

We decided to build inery wallet based on our previous experience.
We analyzed the successful projects and the bellow road map is the result of that analysis.

Custom UX/UI

An essential part of the wallet is UX/UI which lets us visualize INR blockchain data. Not a simple design and will take a lot of time.

INR Coin Support

The major part of this wallet is to Receive and Send INR coins via JSON RPC API :) simple part but need to have a blockchain explorer.

INR Fungible Tokens

Fungible tokens or assets are divisible and non-unique. adding this feature requires a standard Token structure just like the erc-20.

INR Non Fungible Tokens

Non fungible assets are unique and non-divisible. This part may be more difficult rather than fungible tokens because need to ipfs storage or alternative storage.

Human Readable Address

replacing the HRAs used by Inery with a creative and lovely one like MyName.INR-WALLET.com :) for better SEO

Dapp Support

Coding a javascript library for injecting into the web to supporting decentralized applications. hardest part of the road map :)
